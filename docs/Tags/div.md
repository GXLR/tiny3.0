##  Div

### 描述


`<div>` 标签定义TML文档中的分隔（division）或部分（section）。  
`<div>` 标签常用于组合块级元素，以便通过样式表来对这些元素进行格式化。  
div元素采用flex布局。  
dive取消滑动属性，滑动功能采用新控件**scroller**实现。
